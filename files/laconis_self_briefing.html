ContentCore — eigenes CMS
Was ist ContentCore?
Selbst gebautes headless CMS-System, selbst gehostet für alle Kunden. Sauber, intuitiv, ohne Bloat — kein geteilter Server, kein fremdes System, volle Kontrolle.
Features
Mehrsprachigkeit KI / DeepL Übersetzung 4-fach Sicherheit + 2FA Eigene Statistiken (kein GA) Kein Cookie-Banner nötig Newsletter Shop-Bereich Image Optimiser ACF Fields Rollen: Admin / Redakteur
Gamification — Medaillen-System
Basierend auf Website-Aktivität (Besucher, Posts):
🥉 Bronze — Dark Mode 🥈 Silber — eigene Akzentfarbe 🥇 Gold — Gratis-Hosting 💠 Platin — Essen mit Nicolas
+ Steam-ähnliche Achievement-Badges (reine Spielerei, kein Gewinn)
Server-Architektur
Zwei separate VPS (Hostinger) — Cloud-Server für Kunden-Frontends · Separater VPS für ContentCore (Backend/CMS) · Headless: Frontend kommuniziert per API mit ContentCore · Professionelle Trennung der Systeme.
Ich stehe mit meinem Namen für die Performance.
